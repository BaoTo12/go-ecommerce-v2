# Analytics Service

Business intelligence and data aggregation service.

## Purpose
Collects, aggregates, and analyzes business metrics and user behavior data.

## Technology Stack
- **Database**: ClickHouse (OLAP, fast aggregations)
- **API**: gRPC

## Key Features
- ✅ Event tracking (page views, purchases, etc.)
- ✅ Metric recording (revenue, conversions)
- ✅ Real-time dashboards
- ✅ Sales reports
- ✅ User behavior analysis
- ✅ Product performance metrics
- ✅ Funnel analysis
- ✅ Cohort analysis

## Tracked Events
- page_view, product_view, add_to_cart, purchase, search, etc.

## Metrics
- GMV, conversion rate, AOV, ARPU, retention rate, etc.
