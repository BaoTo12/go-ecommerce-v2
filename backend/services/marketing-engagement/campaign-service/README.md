# Campaign Service

Marketing campaign management and execution.

## Purpose
Create, manage, and track marketing campaigns across multiple channels (email, push, SMS, banners).

## Key Features
- ✅ Multi-channel campaigns (email, push, SMS, banner, popup)
- ✅ Audience targeting and segmentation
- ✅ Campaign scheduling
- ✅ Performance metrics (sent, opened, clicked, converted)
- ✅ ROI tracking
- ✅ A/B testing integration
- ✅ Personalization

## API
- `CreateCampaign`: Setup new campaign
- `ActivateCampaign`: Start campaign
- `RecordConversion`: Track campaign ROI
